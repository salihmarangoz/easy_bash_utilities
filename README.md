# easy_bash_utilities

todo description

## Install

```bash
$ git clone git@github.com:salihmarangoz/easy_bash_utilities.git ~/.easy_bash_utilities
```

And add the following line at the end of `~/.bashrc` file.

```
source $HOME/.easy_bash_utilities/easy.bash
```

## Update

```bash
$ cd ~/.easy_bash_utilities
$ git pull --ff-only
```

## Usage

```bash
$ easybash_help
```
